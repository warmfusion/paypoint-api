Capture an authorisation

If you’re happy with the results of any internal checks and you are now in a position to capture the funds for an authorisation then use the Capture request. You can only capture the full amount of the original authorisation and you’ll need to provide the reference for the original authorisation.

Captured authorisations can be Refunded as required using the Refund request.
