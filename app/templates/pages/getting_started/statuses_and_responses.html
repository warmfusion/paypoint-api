Statuses and responses

	1) Blue Processing - Overall
Field Name: Status
Possible values: Success, Failed, Pending, Expired, Cancelled, Voided
API Location: <outcome>
Blue Processing - Detailed
Field Name: Status
Possible Values: Success, Failed, Suspended, Suspended_3DS, Expired, Cancelled, Voided, Suspended_Replay
API Location: <transaction>

	2) Generic Server Processing
Field Name: Status
Possible values: Authorised, Error, Declined
API Location: <authResponse>

	3) PSP Processing
Field Name: Gateway Message
Possible values: Unknown - PSP dependant.
API Location: <authResponse>

	4) Acquirer Processing
Field Name: Message
Possible values: Unknown - Acquirer dependant.
API Location: <authResponse>
